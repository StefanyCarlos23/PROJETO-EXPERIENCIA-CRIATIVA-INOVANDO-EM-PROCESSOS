# Projeto: Experiência Criativa - Inovando em Processos

Projeto desenvolvido na disciplina **Experiência Criativa – Inovando em Processos** (5º semestre).

## Objetivo
Analisar a maturidade dos processos de software de uma empresa, compreender seu funcionamento atual e propor melhorias e automações alinhadas a modelos de qualidade como o MPS.BR, visando maior eficiência, organização e qualidade no desenvolvimento de software.
